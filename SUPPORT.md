Getting Support

Before opening an issue, please check the resources below. Most questions are answered there.


Documentation

Each repository has a README with installation instructions, usage examples, and configuration options. Start there.

Bug Reports and Feature Requests

If you have found a bug or want to request a feature, open an issue in the relevant repository using the appropriate template. Please search existing issues first to avoid duplicates.

General Questions

For questions that are not bugs or feature requests — setup help, usage advice, or general enquiries — reach out at: nivoticslab@gmail.com

For Nivotics Academy Learners

If you are a Nivotics Academy learner working with these tools as part of your programme, reach out through your Academy channels first. Your instructors and mentors are your first point of contact for learning-related questions.

Response Times

We are a small team. We do our best to respond to all enquiries within a reasonable time, but we cannot guarantee response times. Commercial support arrangements are not currently available.
